Smart Library System

Program Smart Library System adalah aplikasi manajemen perpustakaan berbasis C++ yang menggunakan berbagai struktur data untuk mengelola data pengguna dan buku secara efisien. Sistem ini mendukung fitur login & registrasi pengguna (menggunakan hash table), manajemen data buku (dengan circular double linked list), serta fitur tambahan seperti pencarian, pengurutan berdasarkan rating, antrian peminjaman (queue), riwayat pengembalian (stack), dan rekomendasi buku berdasarkan kategori (rekursif).

Virtual Museum Tour Guide

Program ini merupakan simulasi tur virtual di dalam museum berbasis console application yang dibuat menggunakan bahasa pemrograman C++. Aplikasi ini mengintegrasikan berbagai struktur data dan algoritma untuk mensimulasikan sistem interaktif bagi pengguna yang ingin menjelajahi museum secara digital..
